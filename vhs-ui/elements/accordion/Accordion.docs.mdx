A component that does something special and renders text in a div.

### Component usage

```js
<Box width={'400px'}>
  <Accordion>
    <AccordionItem
      title={
        <Box color="white" height={'96px'}>
          A title
        </Box>
      }
      content={
        <Box color="white" height={'96px'}>
          Content
        </Box>
      }
    />
  </Accordion>
</Box>
```
